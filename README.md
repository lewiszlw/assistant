# assistant
个人助手（纯个人使用，旨在解决工作生活中的痛点、提高效率）。

功能
- [x] 每日早安（wechat）
- [x] 即时闹钟（flutter）
- [x] 每日看点
- [ ] 电影推荐

项目结构
- assistant-server: 作为web server提供API
- assistant-script: 爬虫等脚本
- assistant_flutter: flutter客户端
